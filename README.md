# My Pets Heaven Database Management System

## Project Overview

My Pets Heaven is an Oracle SQL and PL/SQL database project designed to manage pet adoption, animal records, shelters, medical treatments, training, donations, payments and customer information.

The project demonstrates relational database design, SQL queries, views, PL/SQL functions, joins, aggregations and formatted reports.

## Key Features

* Customer and animal management
* Pet adoption and consultation tracking
* Medical examination records
* Pet training and trainer management
* Shelter and donation tracking
* SQL views and analytical reports
* PL/SQL functions using `SYS_REFCURSOR`
* Complex joins and aggregate functions

## Technologies Used

* Oracle SQL
* PL/SQL
* Oracle SQL Developer
* Relational Database Design
* SQL Views
* PL/SQL Functions
